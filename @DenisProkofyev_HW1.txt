16) просмотреть содержимое в реальном времени (команда 'grep'), изучите, как она работает.
tail -f test_2/test_4/file1.txt

#Альтернатива - отображение файла через команду less:
less -s -N file1.txt && Shift + F || less +F -s -N file1.txt

# -s - убрать пустые строки, -N - добавить нумерацию строк.

После сохранения изменений в file1.txt через Блокнот они показаны в окне GitBash. 
При редактировании файла через VIM второго терминала изменения не отображаются в реальном времени, через cat >> - отображаются.

17) вывести несколько первых строк из текстового файла

# Команда актуальна при нахождении в директории с файлом. Если остались на 2 уровня выше - по пунктам ДЗ, то:

# Команда актуальна при нахождении в директории с файлом. Если остались на 2 уровня выше - по пунктам ДЗ, то:
tail -n5 test_2/test_4/file1.txt
